openAgile Integrations Take advantage of more than 45 pre-built integration connectors to extend VersionOne and create a single, synchronized agile software development environment. Connectors are available for all VersionOne product editions, or you can build your own using VersionOne’s open, web-service API and SDKs (Java & .NET).

VersionOne Connectors

  • Easily integrate with your existing tools
  • 45+ pre-built integration connectors
  • Integrations available for all editions
  • Extensible API & SDK (.NET and Java)

VersionOne Application Directory

Integrate your applications with VersionOne today. Connectors are available for many commonly used software development tools, including: IDEs; defect trackers; continuous integration, test and source code management tools; portfolio, project and requirements management tools, and more.

Many connectors (including source code for customization) are supported by VersionOne or contributed by the community. Others are available through our network of technology partners.
